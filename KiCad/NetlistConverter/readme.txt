ネットリストコンバータについて

・FlatSpice.py
階層シートを使って階層的に作られた回路図をフラット化し、1つのサブ回路としてネットリストを生成します。


・HierarchicalSpice.py (旧名 SpiceConverter.py)
※使用は非推奨です。具体的には、呼び出した階層シートのピン同士を直接接続している配線がある回路図ではサブ回路のIOが正しく抽出されません。
階層シートを使って階層的に作られた回路図から、階層情報を保持したままネットリストを生成します。



Usage [KiCad (5.1.10)-1 時点]:
1．適当なフォルダにネットリストコンバータ(xxx.py)を置きます。
2．KiCad Eeschemaで"ツール"→"ネットリスト ファイルを生成"をクリックしてネットリストの出力のウィンドウを開きます。
3．"プラグインを追加"をクリックして、"プラグインを参照"から先ほどのネットリストコンバータを選択します。
4. 「ネットリスト コマンド」には自動的に以下のように入力されます。
　python "xxxxx\xxx.py" "%I" "%O"
  "%O"は出力されるファイル名で、デフォルトで拡張子を持たないファイルになるため、次のように変更してネットリストのファイル（.cirファイル）を得るようにします。
　"%O" → "%O.cir"
5. 「タイトル」には「xxx.py」のxxxの部分を入力します。
6．次回以降もこのネットリストコンバータを使うときは「デフォルトの出力形式に設定」にチェックを入れておくと楽です。
7．"ネットリストを生成"をクリックするとプロジェクトフォルダにネットリストのファイルが生成されます。
